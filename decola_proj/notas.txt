Pensamento Computacional
É uma habilidade de formular problemas e encontrar uma solução para eles
Pilares:
	Decomposição: separar o problema em subproblemas
	Reconhecimento de padrões: identificar padrões ou tendencias dentro de um problema
	Abstração: extrapolar o conceito de um problema para uma forma generalista
	Design de algorítimos (automatizar): definir passo a passo a solução de um problema

Raciocínio lógico:
	é a forma do pensamento estruturado que permite encontrar a conclusão ou determinar a resolução de um problema

Decomposição:
	é o primeiro passo na resolução de um problema complexo para resolver um problema complexo, 
	primeiro devemos dividi-lo em problemas menores. 
	Esses problemas menores são mais fáceis de serem resolvidos.

Estratégia:
	Análise: estudar o problema para poder decompô-lo em	partes menores
	Síntese: Combinar os elementos recompondo o problema	original
